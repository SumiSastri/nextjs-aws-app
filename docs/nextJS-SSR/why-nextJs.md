If you want to build production-ready React apps, Next is a good solution. It comes with its own opinions and protocols which depending on your use case you may or may not agree with.

Advantages

1. File based-routing (front-end)
2. Server-side code bundling and minifying (pre-rendering of HTML)
3. Better performance
4. SEO friendly
5. API routes (back-end)
6. CSS modules (can be overwritten)
7. Authentication - multiple patterns for different use cases
8. Production build ready for deploy
